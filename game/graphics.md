|------------------------|-----------------------------------|--------------------------------|----------------------------------|---------------------------------------------------|
| 디스플레이 모드         | Display Mode                      | 디스플레이=screen, 모드=mode   | 창, 전체화면, 전체 창             | Selects windowed, fullscreen, or borderless mode |
| 해상도                 | Resolution                        | 해상도=pixel count             | 출력 화면 크기                   | Sets the output screen resolution                |
| 해상도 스케일           | Resolution Scale                  | 스케일=scale/ratio             | 내부 렌더링 해상도 비율           | Adjusts internal rendering resolution scale      |
| 동적 해상도             | Dynamic Resolution                | 동적=automatic                 | FPS에 따라 해상도 자동 조절       | Automatically adjusts resolution to maintain FPS |
| 동적 해상도 목표 FPS    | Dynamic Resolution Target FPS     | 목표=target                    | 유지하려는 프레임 수              | Target frame rate for dynamic resolution         |
| 업스케일링 방식         | Upscaling Method                  | 업스케일링=upscaling           | 해상도 보정 방식 선택             | Selects upscaling technique                      |
| 계단 현상 방지 방식     | Anti-Aliasing Method              | 계단 현상=aliasing             | 화면 가장자리 부드럽게 처리       | Smooths jagged edges                             |
| 최대 FPS               | Max FPS                           | 최대=maximum                   | 프레임 상한 설정                 | Sets a frame rate limit                          |
| 수직 동기화             | Vertical Sync (V-Sync)            | 수직 동기화=sync               | 화면 찢어짐 방지                 | Prevents screen tearing                          |
