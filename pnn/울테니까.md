-(으)ㄹ 테니까 
Expectation for something to be a certain way

Common sentence structure:
A........ 으면 B....... 을 테니까 C........ 하세요 

김치찌개에 고추가루를 많이 넣으면 너무 매울 테니까 조금만 넣으세요. 
맥주를 많이 마시면 너무 취할 테니까 물도 많이 마시세요 


비가 올 테니까 우산을 가져가세요.
→ It’ll probably rain, so take an umbrella.

길이 많이 막힐 테니까 조금 일찍 출발합시다.
→ The roads will likely be crowded, so let’s leave a bit early.

내가 먼저 도착할 테니까 걱정하지 마세요.
→ I’ll arrive first, so don’t worry.

오늘은 많이 피곤할 테니까 일찍 자는 게 좋아요.
→ You’ll probably be very tired today, so it’s better to go to bed early.

날씨가 추울 테니까 따뜻하게 입으세요.
→ It’ll be cold, so dress warmly.

사람이 많을 테니까 예약을 해 두세요.
→ There will probably be a lot of people, so make a reservation.

지금 가면 늦을 테니까 내일 가요.
→ If we go now, we’ll be late, so let’s go tomorrow.

내가 설명해 줄 테니까 잘 들어요.
→ I’ll explain it, so listen carefully.

배가 고플 테니까 미리 좀 먹어 두세요.
→ You’ll probably be hungry, so eat a little beforehand.

문제가 생길 테니까 백업을 해 두는 게 좋아요.
→ Problems might come up, so it’s good to make a backup.
